make_fact_collection() → FactCollection

Creates a FactCollection from the current graph.

This method deep copies the current graph and assigns unique IDs and empty label lists to each node and edge. It then populates the labels and attributes for nodes and edges, and constructs a list of facts representing these properties.

Returns:

FactCollection: A collection of facts representing the nodes and edges of the graph.
